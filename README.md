# DogFight3
Source code for Dog Fight 3 from 2005.  
A classic Lan multiplayer party game for Windows OS.  
![DogFight3_Image](http://logicwork.net/timeline/sparetime/windows_df32.jpg)

## Libraries used
* OpenGl  (Graphic)  
* WinSock (Network)  
* Fmod    (Sound/Music)
